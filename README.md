This repository contains my Week 1 assignment submission for the Cognizant Digital Nurture 4.0 Java Full Stack Engineer (Java FSE) program. The goal of this week was to strengthen our understanding of fundamental design patterns and algorithmic thinking using Java. The tasks provided include hands-on implementation of core design patterns such as the Singleton Pattern and Factory Method Pattern, which are essential in building reusable and maintainable software. The Singleton Pattern was implemented to ensure that only one instance of a class is created and shared globally, which is a common scenario in logging and configuration management systems. The Factory Method Pattern was demonstrated using a real-world theme—creating different types of document objects like Word, PDF, and Excel—allowing abstraction in object creation based on runtime decisions. In addition to design patterns, we explored algorithmic problem-solving through an E-commerce product search use case. The objective was to search for a product name using both Linear and Binary Search algorithms. Time complexity comparisons were included to highlight performance differences, with binary search clearly standing out for larger datasets due to its logarithmic time complexity. Finally, the week’s assignment concluded with a Financial Forecasting task implemented using recursion. This simulation projected future values based on present values and growth rates, showcasing how recursive functions break a large problem into smaller sub-problems until a base condition is met. Overall, this repository reflects not only my understanding of theoretical concepts but also my ability to apply them in structured Java code. Each exercise is accompanied by its output to validate correctness. This foundational work sets the stage for more advanced full-stack development topics in the upcoming weeks. I have included clean, readable code with comments and structure that adhere to good software engineering practices. I am confident that this work reflects my dedication to mastering both core and applied aspects of Java development, preparing me for enterprise-level project work.
